# Исследование надёжности заёмщиков
[Открыть проект в NBViewer](https://nbviewer.jupyter.org/github/Artemii-Kravtsov/thousands-of-hours/blob/master/1_credit_scoring_leads.ipynb)
<br>

## Задачи 
- Есть ли зависимость между наличием детей и возвратом кредита в срок?
- Есть ли зависимость между семейным положением и возвратом кредита в срок?
- Есть ли зависимость между уровнем дохода и возвратом кредита в срок?
- Как разные цели кредита влияют на его возврат в срок?


## Коротко о данных 
21525 записей. 7 числовых столбцов, 5 категориальных. Пропуски в двух столбцах. Есть грубые дубликаты, есть неявные дубли текстовых категорий. 


## Библиотеки 
`pandas` `pymystem3` `nltk`


## Содержание проекта 
- изучение данных в столбцах
- анализ пропусков
- лемматизация 
- создание сводных таблиц
